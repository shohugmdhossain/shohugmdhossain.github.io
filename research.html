Research Overview

My research integrates Geospatial Artificial Intelligence (GeoAI), remote sensing, and planetary science. I build interpretable deep-learning models and scalable processing workflows to study geomorphology, mineralogy, and urban systems. My work emphasizes reproducible AI for both Earth observation and planetary exploration.

Current Projects (2024–2025)

Project Description Methods / Tools
Mars Mound Detection Using Deep Learning
Planetary Geomorphology | 2024–Present		 Developing Faster R-CNN and Mask R-CNN models for detecting mound-like features on CTX imagery. Using LONI A100 GPU clusters for training, the project enhances morphological mapping of Martian craters. Outcomes include a high-accuracy object-detection model and a curated geospatial dataset of over 5,000 labeled features. Detectron2 · PyTorch · COCO JSON · GPU Training
CRISM Spectral Mineral Mapping
JHU/APL Collaboration | 2024–Present		 Applying autoencoder-based deep unmixing for CRISM hyperspectral data to identify alteration minerals on Mars. The work supports spectral characterization of aqueous environments and improves material classification using AI-driven feature extraction. CRISM · Autoencoders · Spectral Python (SPy) · Unmixing
Urban Accessibility AI (Sidewalk Project)
Remote Sensing & Computer Vision | 2024		 Constructing computer-vision pipelines for detecting sidewalk elements — curb ramps, crosswalks, and traffic-control features — using YOLOv8 and Faster R-CNN. Includes automated label verification through LLM-assisted review, boosting validation accuracy by 10–15%. YOLOv8 · Faster R-CNN · LLM Vision QA · Pandas
